# Eye Fatigue Detection System

This project is a computer-vision-based eye fatigue detection system using a webcam, Python Flask backend, and a web frontend.

## Features

- Real-time webcam-based eye monitoring
- Blink frequency detection
- Viewing distance estimation
- Brightness estimation
- Eye-use time tracking
- Health reminder alerts

## Tech Stack

- Python
- Flask
- OpenCV
- MediaPipe
- HTML
- CSS
- JavaScript
